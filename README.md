Realtime Device Tracker.

To be used locally or in an isolated network.

Hosting Instructions:
1) Clone the repo
2) Open a terminal inside the folder
3) run "npm install"
4) run "npx nodemon"
5) go to 127.0.0.1:[The port provided in the website log code]
